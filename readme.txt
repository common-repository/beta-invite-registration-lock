*NOT READY YET* This plugin will lock down the registration page for your blog, and require a user to enter a passkey in order to view the registration page.

== Description ==

It adds another layer in front of the registration process to help secure your site.

There is an administration section to the plugin to manage and view the current and used passkeys, and re-enable them if you so choose. You can also bulk generate thousands of passkeys. 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
